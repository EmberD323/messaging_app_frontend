## Tech Stack

- **Framework**: React, using Vite.
- **Styling**: CSS
- **Prisma**: As the ORM for interacting with the PostgreSQL database.
- **PostgreSQL**:The database storing user and message data.
- **Supabase**:Image cloud storage.
